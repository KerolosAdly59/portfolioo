# 🧑‍💻 Kerolos Adly — Portfolio

A personal portfolio website built with **Next.js** and **Tailwind CSS**, showcasing my projects, skills, and experience as a Front-End Developer.

---

## 🔗 Live Demo

**[https://portfolioo-theta-taupe.vercel.app](https://portfolioo-theta-taupe.vercel.app)**

---

## ✨ Features

- **Hero Section** — Introduction with title, social links, and CV download
- **About / Skills** — Overview of technical skills and background
- **Projects** — Showcase of real-world projects with live demos and GitHub links
- **Contact Form** — Direct messaging via email
- **Responsive Design** — Fully mobile-friendly layout
- **CV Download & View** — One-click access to resume

---

## 🛠️ Tech Stack

| Technology | Purpose |
|---|---|
| **Next.js 14** | React framework with App Router |
| **Tailwind CSS** | Utility-first styling |
| **Vercel** | Deployment and hosting |
